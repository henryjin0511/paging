Paging API

如何使用/How to use


var paging1 = paging(options);
* 如果不需要使用外部方法，可以不需要var

参数/options

var paging1 = paging(options);

Name Type Default Description
cont String|Object 'pagination' 用来储存分页DOM元素，支持传入ID名称或者原生DOM对象
pages Number 1 分页总数
curr Number 1 当前页码
groups Number 5 显示连续分页数量
first String|Number|Boolean 1 首页显示内容，默认为1，传入false时不显示
last String|Number|Boolean 默认为pages的值 尾页显示内容，默认等于总页数，传入false时不显示
prev String '上一页' 上一页按钮显示内容
next String '下一页' 下一页按钮显示内容
total Boolean false 是否显示总页数，默认不显示
skip Boolean false 是否开启跳页，默认不开启
reset Boolean false 第一次加载是否触发跳页事件，默认为false

事件/Events

paging({pageChange:function(){});

Name Signature Description
pageChange function(){//this对象为当前HTMLDivElement,可通过this.opts获取到当前对象参数列表} 当参数选项中reset设置为true，第一次构建分页元素就会触发分页事件
overPageSkip function(){//this对象为当前HTMLDivElement,可通过this.opts获取到当前对象参数列表} 当skip事件触发或是内部跳页input输入时，页码数超出总页数时触发

方法/Methods

var paging1 = paging();paging1.prev();//上一页

.prev();

paging1.prev();//上一页

.next();

paging1.next();//下一页

.skip(num);
